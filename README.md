# JumpCloud MDM Device Management Lab

## Overview
Configured a cloud-based Mobile Device Management (MDM) environment using JumpCloud to manage users, devices, and security policies.

## Objective
Learn how IT administrators remotely manage company devices, enforce security settings, and control user access.

## Tools Used
- JumpCloud
- Windows Endpoint
- Identity Management
- Device Management

## What I Did
- Created a JumpCloud administrator environment
- Configured a cloud directory for device management
- Explored centralized user and device controls
- Connected endpoint management concepts to real-world IT support

## Real World Scenario
A company hires a new employee and needs to securely configure their laptop.

Instead of manually setting up every device, IT can use MDM solutions to:
- Enroll devices remotely
- Apply security policies
- Manage user access
- Monitor company assets

## Skills Learned
- Mobile Device Management (MDM)
- Identity and Access Management (IAM)
- Endpoint Administration
- Cloud Device Management
- IT Asset Management

## Cybersecurity Connection
MDM improves security by ensuring devices follow company policies before accessing resources.

Examples:
- Enforcing password requirements
- Managing authorized users
- Protecting company devices
